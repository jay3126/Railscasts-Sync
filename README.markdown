# LinkedIn

Ruby script to sync Railscasts episodes from http://railscasts.com/ .
    
## Usage

Download the script.

    # Create Railscasts directory
    $ mkdir Railscasts

    # If behind a proxy server
    $ export http_proxy='http://proxy.example.com:8080/'

    # Run the sync script
    $ ruby railscasts_sync.rb
